# LyraLink AI Wiki

Welcome to the project wiki for LyraLink AI.

## What this project includes

- Web chat experience with account auth, plans, and developer diagnostics.
- API endpoints for chat, dataset search, support, billing, and admin flows.
- Mobile-optimized UI across chat and dashboard pages.
- Optional integrations for Discord, PayPal, Pelican, and Plesk.

- Never commit production secrets.
- Use .env values from .env.example as your baseline.
- Rotate keys immediately if exposure is suspected.
